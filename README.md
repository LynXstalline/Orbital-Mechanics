# Satellites, Stewardship, and the Order of the Universe
### An Interactive Physics & Ethics Simulation

This project is a multi-disciplinary exploration of orbital mechanics, calculus-based motion, and the ethical responsibilities of space exploration. It was developed as a part of the **Group 2 PBL (Project-Based Learning) for Class 12B**.

## 🚀 Live Demo
**[Click here to view the interactive simulation](https://LynXstalline.github.io/Orbital-Mechanics/)**

## 🛠️ Features
- **Orbital Physics Engine:** A real-time simulation using the Canvas API to model gravitational pull ($F = G \frac{m_1 m_2}{r^2}$) and tangential velocity.
- **Dynamic Data Visualization:** Live tracking of gravitational force, speed, and orbital status (Stable, Sub-orbital, or Escaping).
- **Interactive Controls:** Users can manipulate orbital radius and velocity to observe real-time effects on satellite trajectories.
- **Multi-disciplinary Content:** Integrated sections covering the Calculus of Motion (rates of change) and the Ethics of Space Debris (Stewardship).

## 💻 Technical Stack
- **Frontend:** HTML5, CSS3 (Custom SVG animations and responsive design).
- **Engine:** JavaScript (Canvas API for physics rendering).
- **Design:** Modern "Dark Space" UI with Orbitron typography and glowing neon aesthetics.
